🏆 Pokémon Champions Battle Meta API

Competitive battle metadata for Pokémon Champions, updated daily at 2 AM UTC.

⏳ Status: Awaiting first scrape. Data will be available after the next scheduled run.

Endpoints

GET battle_meta.json

Full competitive metadata including Pokémon usage stats, top moves, items, abilities, teammates, and tier lists.

Per-Pokémon Data

GET pokemon/{dex_id}.json

Individual Pokémon competitive data by National Dex ID.

Examples: pokemon/445.json (Garchomp), pokemon/448.json (Lucario)

Schema

{
  "schema_version": "1.0.0",
  "updated_at": "2026-04-14T02:00:00Z",
  "season": { "id": "s1", "name": "Season 1", ... },
  "pokemon_usage": [
    {
      "rank": 1,
      "dex_id": 445,
      "name": "Garchomp",
      "usage_rate": 0.342,
      "win_rate": 0.528,
      "top_moves": [...],
      "top_items": [...],
      "top_teammates": [...]
    }
  ],
  "tier_list": { "S": [...], "A": [...], ... },
  "sources": [...]
}

Usage

// Fetch and use in your app
fetch('https://eurekaffeine.github.io/pokemon-champions-scraper/battle_meta.json')
  .then(res => res.json())
  .then(data => {
    console.log(`${data.pokemon_usage.length} Pokémon loaded`);
    console.log('Top Pokémon:', data.pokemon_usage[0].name);
  });

Caching

Data is updated daily. Use If-Modified-Since or ETag headers for efficient caching:

fetch(url, {
  headers: { 'If-Modified-Since': lastModified }
}).then(res => {
  if (res.status === 304) return cachedData; // Not modified
  return res.json();
});
