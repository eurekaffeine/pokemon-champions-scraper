🏆 Pokémon Champions
Battle Meta API

Competitive battle metadata for mobile apps, updated daily.

📡 API Endpoints

GET battle_meta.json

Returns full competitive metadata including:

  • Pokémon usage rankings and rates
  • Top moves, items, abilities per Pokémon
  • Top teammates and team compositions
  • Tera type distribution
  • Tier lists (S/A/B/C)
  • Rank distribution stats
GET pokemon/{dex_id}.json

Individual Pokémon competitive data. Examples:

  • pokemon/445.json — Garchomp
  • pokemon/149.json — Dragonite
  • pokemon/448.json — Lucario

💻 Usage Example

// Fetch battle meta
const response = await fetch('https://your-domain/battle_meta.json');
const data = await response.json();

// Top 10 Pokémon by usage
const top10 = data.pokemon_usage.slice(0, 10);
console.log(top10.map(p => `${p.rank}. ${p.name}: ${(p.usage_rate * 100).toFixed(1)}%`));

// Get specific Pokémon details
const garchomp = await fetch('https://your-domain/pokemon/445.json').then(r => r.json());
console.log(garchomp.competitive.top_moves);

Update Schedule
